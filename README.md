# Kyle's Game

A portable Windows game project 

## Download

Download the latest portable Windows build from the repository's **Releases** page.

## Running the game

1. Download the latest `.exe` from Releases.
2. Save it anywhere on your Windows PC.
3. Double-click the `.exe` to launch the game.

No installer is required.

## Project status

This repository is prepared for source code and portable release builds. Add the game source files to the repository, then package the Windows executable and attach it to a GitHub Release.
